# <a name="organization-tax-information"></a>组织税收信息

**适用于**

-  合作伙伴中心

适用于企业的 Windows 应用商店购买的税收由你的企业地址确定。 位于以下国家/地区的企业可以提供其增值税编号或本地等效项：

### <a name="add-your-vat-id"></a>添加增值税 ID

1.  从**仪表板**转到**成员产品/服务**-> **可用**
2.  选择**添加增值税 ID**
3.  输入你的组织的增值税 ID，选择**添加**，然后选择**下一步**。
