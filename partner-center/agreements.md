# <a name="microsoft-cloud-agreements-by-region-and-language"></a>按地区和语言划分的 Microsoft 云协议

**适用于**

- 云解决方案提供商合作伙伴

客户必须接受并签署 Microsoft 云协议, 然后才能代表客户订购。

>[!IMPORTANT]
> 从2018年11月7日开始, 你将需要通过提供有关对其进行签名的用户的信息来确认客户已接受 Microsoft 云协议。 有关更多详细信息, 请参阅[确认客户接受 Microsoft 云协议](confirm-consent.md)。

Microsoft 云协议是区域特定的, 可使用多种语言。 在以下列表之一中查找客户的位置和首选语言, 下载正确的协议。

如果你不确定可以在哪些区域和市场中销售, 请在此列表中查找你公司的位置:[云解决方案提供商计划地区市场和货币](regional-authorization-overview.md)。
