# <a name="participate-in-software-assurance-programs"></a>参与软件保障计划

包括培训和部署服务的软件保障程序是很好地发展你的业务。 Microsoft 进行补偿为其企业客户提供培训、 规划、 部署和其他保障服务的合作伙伴。 你可以在这些程序通过合作伙伴中心注册并可以管理、 保留，并兑换以及在合作伙伴中心使用凭证验证和兑换工具你凭证。
