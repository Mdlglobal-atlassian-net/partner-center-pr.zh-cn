# 来自 Microsoft 的客户通讯


一般情况下，我们希望与客户的所有通讯均来自云解决方案提供商合作伙伴。 由于你具有客户结算关系，Microsoft 不会将有关他们的订阅的通讯直接发送给你的客户。

Microsoft 可能会就有关服务事件或其他操作信息与客户直接通讯。

对于与 Microsoft 已存在现有商业关系的客户，Microsoft 将针对非云解决方案提供商产品和服务与客户直接通讯。
