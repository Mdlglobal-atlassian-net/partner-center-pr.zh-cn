# <a name="reset-your-password"></a>“重置密码”

如果您尝试登录到合作伙伴中心，并且你不记得密码，请执行以下操作：

1. 输入你的用户名后, 在下一屏幕中选择**忘记了密码**。

2. 你将需要再次登录。 提供电子邮件、 电话或 Skype，并选择选项之一：

- “创建帐户”

- 无法访问帐户

- 登录选项

3. 选择**无法访问帐户**。
4. 选择**工作或学校帐户**并回答的问题。
