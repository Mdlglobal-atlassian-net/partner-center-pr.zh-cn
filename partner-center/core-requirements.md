# <a name="core-requirements-for-incentives-co-op-claims"></a>奖励合作申请的核心要求

**适用对象**

- 合作伙伴中心

提交奖励申请的每种类型的活动都有需要满足的核心要求，你需要在执行证明文档中展示这些要求的满足情况。 在某些情况下，还需要一些已定义的通信类型。

下表概述了每种可接受活动的核心要求和通信要求。
