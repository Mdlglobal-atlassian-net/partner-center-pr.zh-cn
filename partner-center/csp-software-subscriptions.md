# <a name="sell-software-subscriptions-through-csp"></a>CSP를 통해 소프트웨어 구독 판매

**적용 대상**

- 云解决方案提供商

**적절한 역할**

- 관리 에이전트
- 전역 관리자

借助 Azure 保留和服务器订阅（Windows Server 和 SQL Server 订阅），CSP 计划中的合作伙伴可以更好地满足快速增长的客户需求，以实现可高度预测性和持久性的更经济高效的解决方案云工作负荷。 

你现在可以通过利用 Azure 混合权益，通过合作伙伴中心和 Azure 门户代表商业客户获取、预配和管理 Azure 预订和服务器订阅。

Azure 하이브리드 혜택은 Windows Server 라이선스에서 더 많은 가치를 창출하고 가상 컴퓨터에서 최대 40%의 비용을 절감할 수 있도록 도와줍니다. Software Assurance에서 적용되는 Windows Server Datacenter 및 Standard 버전 라이선스를 통해 이 혜택을 이용할 수 있습니다. 根据版本，你可以在 Azure 中转换或重新使用许可证，以在 Azure 中运行 Windows Server 虚拟机，并按较低的基础计算费率（例如，Linux 虚拟机速率）。

## <a name="buy-software-subscriptions-on-behalf-of-customers"></a>고객을 대신하여 소프트웨어 구독 구입

代表客户购买软件订阅：

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 파트너 센터 메뉴에서 **고객**을 선택한 다음, 목록에서 고객을 선택합니다.

3. 在客户的详细信息页中，选择 "**添加产品**"，然后按照屏幕上的说明创建订单并为订单付费。 除澳大利亚和巴西以外，所有商业定价都不包括税款。 对于澳大利亚和巴西，价格包含税款。

## <a name="activate-and-manage-software-subscriptions"></a>소프트웨어 구독 정품 인증 및 관리

购买软件订阅后，请按照以下步骤进行下载。

>[!NOTE]
>你必须是管理员代理才能下载软件和获取激活密钥。

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 파트너 센터 메뉴에서 **고객**을 선택한 다음, 목록에서 고객을 선택합니다.

3. 고객의 세부 정보 페이지로 이동하여 **소프트웨어**를 선택합니다. 你将看到代表客户购买的所有软件的列表。 

4. 다운로드할 제품을 펼칩니다. **제품 선택** 필드에서 원하는 **버전**, **언어** 및 **파일 유형/OS**를 선택합니다. 

5. **제출**을 선택하여 특정 제품을 표시합니다. 

6. **키 및 다운로드 받기**를 선택합니다. 

7. **다운로드**를 선택하여 다운로드를 시작하거나 **링크 복사**를 선택하여 링크를 복사해 고객에게 전송합니다. 

>[!NOTE]
>이 링크는 2주 후 또는 다운로드 50회 이후(둘 중 빠른 시간 기준)에 만료됩니다. 링크가 만료될 때 이 페이지로 돌아가서 **키 및 다운로드 받기**를 다시 선택하면 또 한 번 2주 또는 다운로드 50회의 기간이 주어집니다. 필요한 횟수 만큼 이러한 작업을 수행할 수 있습니다. 

## <a name="view-activity-for-software-key-access-and-software-downloads"></a>查看软件密钥访问和软件下载的活动

出于审核或符合性目的，可能需要检查具有已访问的服务器订阅软件密钥或下载的服务器订阅软件的用户的列表。 使用以下过程来访问此信息。 

>[!NOTE]
>只有全局管理员、帐户管理员、引用管理员或营销内容管理员才能看到这些活动日志。 

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 选择右上角的齿轮图标。

3. 在菜单中，选择 "**活动日志**"。

4. 输入要查看的活动的日期范围。 "活动日志" 将显示在指定的时间内具有已访问的软件密钥或已下载软件的用户的列表。 

## <a name="cancel-a-purchase"></a>구입 취소

你可以在购买日期的60天内取消软件购买。 如果在第一个60天内取消，则不会向你收取提前终止费。 60天后，你将无法再取消购买。 （请参阅此取消规则的主要限制。 若要了解取消软件购买后会发生的情况，请参阅这些步骤后面的重要说明。） 

>[!NOTE]
>用于取消购买的以下步骤仅适用于在特定取消时段（例如在购买后的前60天内）限定取消的软件。 请注意，取消购买的这些步骤不适用于购买 Microsoft Windows 7 扩展安全更新2020。 （若要取消 Windows 7 ESU 2020，你需要提交客户支持请求，如本主题后面所述。）这些步骤也不适用于 Azure 中的 SUSE Linux 或 RedHat 软件计划。 目前，无法取消或交换 SUSE 或 RedHat 软件计划。 [了解](https://docs.microsoft.com/azure/virtual-machines/linux/prepay-suse-software-charges)有关使用 SUSE Linux 或 RedHat 计划的详细信息。

请按照以下步骤取消购买：

>[!NOTE]
>관리 에이전트여야 구입을 취소할 수 있습니다. 以下步骤介绍了如何取消合作伙伴中心仪表板中的购买。 也可以使用[合作伙伴中心 API](https://docs.microsoft.com/partner-center/develop/cancel-software-purchases)来实现此目的。

1. 在开始取消过程之前，请确保具有以下各项：

    - 客户的名称、租户 GUID 或域名

    - 要取消的产品的名称
    
    - 订单 ID

2. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

3. 파트너 센터 메뉴에서 **고객**을 선택한 다음, 목록에서 고객을 선택합니다.

4. 在客户的详细信息页上，选择 "**软件**" 以查看为客户购买的软件的列表。 

5. 找到要取消的软件购买，然后选择 "**取消**"。 随即出现一个对话框。

6. 从 "订单号" 下拉列表中，选择要取消的正确订单 ID 号。 （您可以从客户的**订单历史记录**页中了解有关订单或订单 ID 号的详细信息。）

7. 选中相应的复选框以确认您已阅读有关取消的**重要**消息。 （请参阅下面的**重要**说明，详细了解取消购买后会发生的情况。）

8. 选择 "**提交**" 以取消购买。 如果要为客户取消多个订单，则需要对每个唯一的订单 ID 号再次执行步骤4到步骤6。

当你尝试取消订单时，合作伙伴中心还可能会向你显示其他信息（显示在 "订单号" 下拉列表下）。 此信息可能包括：

* 你还剩多少天来取消该特定订单

* 是否已传递取消窗口，并且无法再取消订单

* 如果我们需要有关你的取消请求的详细信息，你可能会获得一个指向**客户支持请求**表单的链接。

>[!IMPORTANT]
>取消订单后，将显示一条确认取消的消息。 但在 "合作伙伴中心" 仪表板上显示取消之前，可能会有最多15分钟的延迟。 另请注意以下 "取消后的详细信息"。

取消购买后：

- 将吊销所有相关的软件密钥和下载链接。 这意味着，你和你的客户都无法使用软件密钥和下载任何与此次购买相关的链接。 你和你的客户负责停止所有已取消的软件的使用。 你还负责卸载已取消的软件，并删除所有相关的软件下载和链接。

- 已取消的项仍将显示在客户的软件详细信息页上，但激活密钥将不可用。

- 将在下一个月度发票上显示已取消订单的信用额度。 永久性软件将收到100% 的信用额度，软件订阅将获得按比例计费。

### <a name="submit-a-customer-support-request-to-cancel-a-purchase"></a>提交客户支持请求以取消购买

如果你尝试通过合作伙伴中心取消软件购买，但被告知提供详细信息并填写客户支持请求表，则这些步骤可能会有所帮助：

1. 当你从 "取消购买" 窗口中选择 "**客户支持请求**" 链接时，将打开 "合作伙伴中心的问题" 页面。

2. 在 "详细信息" 下的 "问题类型" 列表中，选择**代表客户购买/退款 CSP**。

3. 填写 "影响" 和 "标题" 字段。

4. 在 "描述" 字段中，提供以下内容：

    - 고객 테넌트 GUID 또는 도메인 이름
    
    - 주문 ID 또는 구독 ID
    
    - 환불 이유

    - 요청된 금액

5. 在 "联系人" 字段中，输入你的姓名、电子邮件地址和电话号码。

6. 어떤 이유로든 파일을 첨부해야 하는 경우에는 **파일 추가**를 선택합니다. 이 단계는 선택 사항입니다.

7. 完成后，选择 "**提交**"。
