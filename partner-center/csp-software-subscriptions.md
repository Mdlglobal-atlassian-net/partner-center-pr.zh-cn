# <a name="sell-software-subscriptions-through-csp"></a>通过云解决方案提供商销售软件订阅

**适用于**

- 云解决方案提供商

**相应的角色**

- 管理员代理
- 全局管理员

借助 Azure 保留和服务器订阅（Windows Server 和 SQL Server 订阅），CSP 计划中的合作伙伴可以更好地满足快速增长的客户需求，以实现可高度预测性和持久性的更经济高效的解决方案云工作负荷。 

你现在可以通过利用 Azure 混合权益，通过合作伙伴中心和 Azure 门户代表商业客户获取、预配和管理 Azure 预订和服务器订阅。

Azure 混合权益可帮助你从 Windows Server 许可证中获取更多价值，最多可节省 40% 的虚拟机费用。 你可以通过软件保障所涵盖的 Windows Server Datacenter 和 Standard Edition 许可证来使用此权益。 根据版本，你可以在 Azure 中转换或重新使用许可证，以在 Azure 中运行 Windows Server 虚拟机，并按较低的基础计算费率（例如，Linux 虚拟机速率）。

## <a name="buy-software-subscriptions-on-behalf-of-customers"></a>代表客户购买软件订阅

代表客户购买软件订阅：

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 在“合作伙伴中心”菜单中，选择“客户”，然后从列表中选择客户。

3. 在客户的详细信息页中，选择 "**添加产品**"，然后按照屏幕上的说明创建订单并为订单付费。 除澳大利亚和巴西以外，所有商业定价都不包括税款。 对于澳大利亚和巴西，价格包含税款。

## <a name="activate-and-manage-software-subscriptions"></a>激活和管理软件订阅

购买软件订阅后，请按照以下步骤进行下载。

>[!NOTE]
>你必须是管理员代理才能下载软件和获取激活密钥。

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 在“合作伙伴中心”菜单中，选择“客户”，然后从列表中选择客户。

3. 转到客户的详细信息页面，然后选择**软件**。 你将看到代表客户购买的所有软件的列表。 

4. 展开要下载的产品。 在**选择产品**字段中，选择所需的**版本**、**语言**和**文件类型/操作系统**。 

5. 选择**提交**以显示特定产品。 

6. 选择**获取密钥和下载**。 

7. 选择**下载**开始下载，或选择**复制链接**以复制链接并将其发送给客户。 

>[!NOTE]
>此链接将在两周或 50 次下载后（以先到者为准）过期。 链接过期后，请返回此页面并再次选择**获取密钥和下载**，以启用另一个两周或 50 次下载。 可以根据需要多次执行此操作。 

## <a name="view-activity-for-software-key-access-and-software-downloads"></a>查看软件密钥访问和软件下载的活动

出于审核或符合性目的，可能需要检查具有已访问的服务器订阅软件密钥或下载的服务器订阅软件的用户的列表。 使用以下过程来访问此信息。 

>[!NOTE]
>只有全局管理员、帐户管理员、引用管理员或营销内容管理员才能看到这些活动日志。 

1. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

2. 选择右上角的齿轮图标。

3. 在菜单中，选择 "**活动日志**"。

4. 输入要查看的活动的日期范围。 "活动日志" 将显示在指定的时间内具有已访问的软件密钥或已下载软件的用户的列表。 

## <a name="cancel-a-purchase"></a>取消购买

你可以在购买日期的60天内取消软件购买。 如果在第一个60天内取消，则不会向你收取提前终止费。 60天后，你将无法再取消购买。 （请参阅此取消规则的主要限制。 若要了解取消软件购买后会发生的情况，请参阅这些步骤后面的重要说明。） 

>[!NOTE]
>用于取消购买的以下步骤仅适用于在特定取消时段（例如在购买后的前60天内）限定取消的软件。 这些步骤也不适用于 Azure 中的 SUSE Linux 或 RedHat 软件计划。 目前，无法取消或交换 SUSE 或 RedHat 软件计划。 [了解](https://docs.microsoft.com/azure/virtual-machines/linux/prepay-suse-software-charges)有关使用 SUSE Linux 或 RedHat 计划的详细信息。

请按照以下步骤取消购买：

>[!NOTE]
>必须是管理员代理才能取消购买。 以下步骤介绍了如何取消合作伙伴中心仪表板中的购买。 也可以使用[合作伙伴中心 API](https://docs.microsoft.com/partner-center/develop/cancel-software-purchases)来实现此目的。

1. 在开始取消过程之前，请确保具有以下各项：

    - 客户的名称、租户 GUID 或域名

    - 要取消的产品的名称
    
    - 订单 ID

2. 登录到合作伙伴中心[仪表板](https://partner.microsoft.com/dashboard)。

3. 在“合作伙伴中心”菜单中，选择“客户”，然后从列表中选择客户。

4. 在客户的详细信息页上，选择 "**软件**" 以查看为客户购买的软件的列表。 

5. 找到要取消的软件购买，然后选择 "**取消**"。 随即出现一个对话框。

6. 从 "订单号" 下拉列表中，选择要取消的正确订单 ID 号。 （您可以从客户的**订单历史记录**页中了解有关订单或订单 ID 号的详细信息。）

7. 选中相应的复选框以确认您已阅读有关取消的**重要**消息。 （请参阅下面的**重要**说明，详细了解取消购买后会发生的情况。）

8. 选择 "**提交**" 以取消购买。 如果要为客户取消多个订单，则需要对每个唯一的订单 ID 号再次执行步骤4到步骤6。

当你尝试取消订单时，合作伙伴中心还可能会向你显示其他信息（显示在 "订单号" 下拉列表下）。 此信息可能包括：

- 你还剩多少天来取消该特定订单

- 是否已传递取消窗口，并且无法再取消订单

- 如果我们需要有关你的取消请求的详细信息，你可能会获得一个指向**客户支持请求**表单的链接。

>[!IMPORTANT]
>取消订单后，将显示一条确认取消的消息。 但在 "合作伙伴中心" 仪表板上显示取消之前，可能会有最多15分钟的延迟。 另请注意以下 "取消后的详细信息"。

取消购买后：

- 将吊销所有相关的软件密钥和下载链接。 这意味着，你和你的客户都无法使用软件密钥和下载任何与此次购买相关的链接。 你和你的客户负责停止所有已取消的软件的使用。 你还负责卸载已取消的软件，并删除所有相关的软件下载和链接。

- 已取消的项仍将显示在客户的软件详细信息页上，但激活密钥将不可用。

- 将在下一个月度发票上显示已取消订单的信用额度。 永久性软件将收到100% 的信用额度，软件订阅将获得按比例计费。

### <a name="submit-a-customer-support-request-to-cancel-a-purchase"></a>提交客户支持请求以取消购买

如果你尝试通过合作伙伴中心取消软件购买，但被告知提供详细信息并填写客户支持请求表，则这些步骤可能会有所帮助：

1. 当你从 "取消购买" 窗口中选择 "**客户支持请求**" 链接时，将打开 "合作伙伴中心的问题" 页面。

2. 在 "详细信息" 下的 "问题类型" 列表中，选择**代表客户购买/退款 CSP**。

3. 填写 "影响" 和 "标题" 字段。

4. 在 "描述" 字段中，提供以下内容：

    - 客户租户 GUID 或域名
    
    - 订单 ID 或订阅 ID
    
    - 退款原因

    - 请求的金额

5. 在 "联系人" 字段中，输入你的姓名、电子邮件地址和电话号码。

6. 如果出于任何原因需要附加文件，请选择**添加文件**。 此步骤可选。

7. 完成后，选择 "**提交**"。
