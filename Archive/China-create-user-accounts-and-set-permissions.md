<span data-ttu-id="6f8af-104">**适用于**</span><span class="sxs-lookup"><span data-stu-id="6f8af-104">**Applies to**</span></span>

<span data-ttu-id="6f8af-106">为需要在合作伙伴中心上执行任务的员工创建用户帐户。</span><span class="sxs-lookup"><span data-stu-id="6f8af-106">Create user accounts for your employees who need to perform tasks on Partner Center.</span></span> <span data-ttu-id="6f8af-107">仅全局或用户管理管理员可以添加用户和分配角色。</span><span class="sxs-lookup"><span data-stu-id="6f8af-107">Only global or user management admins can add users and assign roles.</span></span> <span data-ttu-id="6f8af-108">在中**帐户设置** &gt; **用户管理**，可以添加用户帐户和分配或更改权限。</span><span class="sxs-lookup"><span data-stu-id="6f8af-108">In **Account Settings** &gt; **User management**, you can add user accounts and assign or change permissions.</span></span>

<span data-ttu-id="6f8af-109">**添加新用户**</span><span class="sxs-lookup"><span data-stu-id="6f8af-109">**Add a new user**</span></span>

1.  <span data-ttu-id="6f8af-110">在合作伙伴中心，转到仪表板菜单&gt;**帐户设置** &gt; **用户管理**。</span><span class="sxs-lookup"><span data-stu-id="6f8af-110">In Partner Center, go to the Dashboard menu &gt; **Account settings** &gt; **User management**.</span></span>

    ><span data-ttu-id="6f8af-118">**重要须知**</span><span class="sxs-lookup"><span data-stu-id="6f8af-118">**Important**</span></span><br><span data-ttu-id="6f8af-119">新用户首次登录合作伙伴中心时，必须使用临时密码登录。</span><span class="sxs-lookup"><span data-stu-id="6f8af-119">When new users sign in to Partner Center for the first time, they must sign in with their temporary password.</span></span> <span data-ttu-id="6f8af-120">确保记录新用户的临时密码，以便在以后发送该密码。</span><span class="sxs-lookup"><span data-stu-id="6f8af-120">Be sure to make a note of the new user's temporary password so you can send it later.</span></span> <span data-ttu-id="6f8af-121">你将无法再次访问此信息。</span><span class="sxs-lookup"><span data-stu-id="6f8af-121">You won't be able to access this information again.</span></span> 

### <a href="" id="setuserpermissions"></a><span data-ttu-id="6f8af-122">设置用户权限</span><span class="sxs-lookup"><span data-stu-id="6f8af-122">Set user permissions</span></span>

<span data-ttu-id="6f8af-123">合作伙伴中心访问权限基于角色，这意味着可以将用户视图限制为仅显示用户完成特定任务所需的特性和功能。</span><span class="sxs-lookup"><span data-stu-id="6f8af-123">Partner Center access is role-based, meaning that you can restrict the user view to show only the features and functions that a user needs to complete specific tasks.</span></span> <span data-ttu-id="6f8af-124">对于每个用户，你必须选择两个设置：</span><span class="sxs-lookup"><span data-stu-id="6f8af-124">For each user, you must select two settings:</span></span>

    ><span data-ttu-id="6f8af-127">**注意**</span><span class="sxs-lookup"><span data-stu-id="6f8af-127">**Note**</span></span><br><span data-ttu-id="6f8af-128">默认设置应**没有管理员**在大多数情况下，除非该员工需要权限才能完成特定的管理任务。</span><span class="sxs-lookup"><span data-stu-id="6f8af-128">The default setting should be **No admin** in most cases, unless the employee requires permissions to complete specific administrative tasks.</span></span>
